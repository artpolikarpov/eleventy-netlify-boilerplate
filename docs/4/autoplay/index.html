Let Fotorama play photos automatically using data-autoplay="true":

<div class="fotorama"
     data-autoplay="true">
  <img src="1.jpg">
  <img src="2.jpg">
</div>

By default, a pause between images is 5 seconds. Set any interval in milliseconds, for example, 3 seconds: data-autoplay="3000".
